# InspectPro — Support

## About

InspectPro is an offline toolkit for refinery inspection and corrosion engineers. It covers welding procedures, coating selection, corrosion rate calculations, fitness-for-service screening, damage mechanism reference, field inspection notes, and more.

## Frequently Asked Questions

**Does the app require an internet connection?**  
No. InspectPro works entirely offline. All data and calculations are performed locally on your device.

**Where is my data stored?**  
All inspection notes, readings, and photos are stored locally on your device. Nothing is uploaded to any server.

**Can I export my inspection reports?**  
Yes. You can export inspection reports as PDF files and share them via email, messaging apps, or any other sharing method available on your device.

**Which API standards does the app cover?**  
API 510, API 570, API 571, API 579, API 582, API 577, API 650, API 653, ASME Section VIII, and ASME B31.3.
